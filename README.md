# 📚 Sistema Aprenda & Calcule

## 📖 Descrição

O **Sistema Aprenda & Calcule** é uma aplicação desenvolvida em **Java** com o objetivo de praticar os conceitos de **Programação Orientada a Objetos (POO)**.

O sistema possui duas funcionalidades principais:

- 🧮 Uma calculadora capaz de realizar operações matemáticas básicas.
- 📚 Um guia de conceitos que apresenta definições sobre Programação Orientada a Objetos.

Além das funcionalidades, o projeto demonstra a utilização de classes, objetos, construtores, encapsulamento, métodos, estruturas de decisão e repetição.

---

## 🚀 Funcionalidades

### 🧮 Calculadora

Permite realizar as operações:

- Soma
- Subtração
- Multiplicação
- Divisão
- Validação para impedir divisão por zero

### 📚 Guia de Conceitos

O sistema apresenta explicações sobre:

- Programação Orientada a Objetos (POO)
- Classes
- Atributos
- Encapsulamento
- Requisitos e Regras de Negócio

## 📄 Classe `Calculadora`

A classe `Calculadora` representa uma calculadora simples.

### Atributos

- `num1`
- `num2`

Os atributos são privados para garantir o encapsulamento.

### Métodos

- `somar()`
- `subtrair()`
- `multiplicar()`
- `dividir()`

Também possui:

- Construtor
- Getters
- Setters
